# Cursor Clone Project

This project is a recreation of the Cursor website, focusing on pixel-perfect implementations of its key sections using HTML and CSS.

## Recreated Sections

The following sections have been rebuilt to match the original design:

*   **Navigation Bar**: Top navigation with logo, links, and action buttons.
*   **Hero Section**: Main landing area with headline, download CTA, and hero image.
*   **Logo Section**: Grid of trusted company logos.
*   **Feature Sections**: Detailed feature highlights (Agent, Tab, Ecosystem) with alternating layouts.
*   **Testimonials**: Grid of user testimonials with author details.
*   **Use Cases**: Cards showcasing different models, search, and enterprise features.
*   **Changelog**: A timeline-style list of recent updates.
*   **About Section**: Information about the Cursor team with a side-by-side layout.
*   **Recent Highlights**: News and blog post highlights.
*   **Final CTA**: Bottom download call-to-action.
*   **Footer**: Comprehensive footer with links, theme toggles, and legal info.

## Design System

### Fonts
*   **Primary Font**: `CursorGothic` (Custom font used for headings and primary text).
*   **Secondary Font**: System fonts (Sans-serif) used for smaller details or fallbacks.

### Font Sizes
*   **Headings**: `36px` (Section Titles), `26px` (Sub-section Titles), `22px` (Feature/Card Titles).
*   **Body Text**: `16px` (Standard descriptions, Links), `15px` (Testimonials).
*   **Small Text**: `14px` (Metadata, Footer links), `13px` (Badges, Tertiary info).

### Color Palette

**Backgrounds**
*   `#14120b`: Main page background (Deep dark brown/black).
*   `#1b1913`: Card and section backgrounds (Slightly lighter dark tone).
*   `#0c0c0c`: About section background (Dark black).

**Text**
*   `#ffffff` / `rgb(237, 236, 236)`: Primary headings and body text.
*   `rgba(237, 236, 236, 0.6)`: Secondary text, descriptions, and metadata.
*   `#888888`: Muted text (social handles, tertiary info).

**Accents**
*   `#ff4d00`: Primary accent color (Links, arrows, hover states).
*   `#e0e0e0`: Button hover states.
